# CPE178_E01_ProjectProposal_Group02_3T2526

## Project Description

The AI Spending Habit Analyzer is a web-based application designed to help users track their daily expenses and predict their overspending risk using the Random Forest machine learning algorithm. The system analyzes users' spending behavior and provides personalized financial recommendations to encourage better budgeting habits.

## Features

- User Registration
- User Login
- Expense Management
- Dashboard
- Spending Analytics
- AI Overspending Prediction
- Personalized Recommendations
